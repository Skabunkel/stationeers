# stationeers

A repository of all the IC10 code im writing for stationeers.
its basiclly MIPS assembly.

## clock.ic10 
A 24 hour clock with hours and minutes.

## lights.ic10
Simple code to turn on an turn off the lights in a green house.

## autoEmpty.ic10
Empties 3 Centrifuges when they are full and restarts them.

## solarPanel.ic10
Automatic 2 axis tracking for solar pannels.
